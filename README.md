# Gestión de Colonias Felinas 🐱

Aplicación web para que los voluntarios que gestionan colonias felinas urbanas puedan tener toda la información organizada en un solo sitio.

## Descripción

Los voluntarios que cuidan colonias felinas necesitan registrar información sobre cada gato: si está esterilizado, testado, enfermo, embarazado, y sus desparasitaciones. Esta app centraliza toda esa información de forma sencilla y accesible.

## Tecnologías

- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: Express (Node.js) + TypeScript
- **Almacenamiento**: LocalStorage
