# Idea del Proyecto: Gestión de Colonias Felinas

---

## 1. Descripción de la idea

Una aplicación web para que los voluntarios que gestionan colonias felinas urbanas puedan tener toda la información organizada en un solo sitio. Actualmente estos voluntarios gestionan los datos de las colonias y los gatos mediante papel, grupos de WhatsApp o Excel, lo que hace difícil mantener la información actualizada y accesible.

---

## 2. Problema que resuelve

Los voluntarios que cuidan colonias felinas urbanas no tienen una herramienta específica para registrar y consultar la información de los gatos. Necesitan saber qué gatos están esterilizados, cuáles están enfermos, cuándo fue la última desparasitación, etc. Sin una app, esta información se pierde o se desorganiza fácilmente.

---

## 3. Usuario objetivo

Voluntarios que gestionan colonias felinas urbanas. Son ellos quienes realizan todas las acciones: alimentar a los gatos, llevarlos al veterinario, esterilizarlos, desparasitarlos y mantener el registro de cada animal.

---

## 4. Funcionalidades principales

- Ver un listado de todas las colonias
- Ver los gatos de cada colonia
- Ficha de cada gato (nombre, color, foto, sexo, edad aproximada)
- Marcar si un gato está esterilizado
- Marcar si un gato está testado (FIV/FELV)
- Marcar si un gato está enfermo y de qué
- Marcar si una gata está embarazada
- Registrar desparasitaciones (fecha y producto utilizado)

---

## 5. Funcionalidades opcionales

- Añadir fotos a la ficha del gato
- Filtrar gatos por estado (esterilizado, enfermo, embarazada...)
- Registrar el cuidador responsable de cada colonia
- Exportar los datos de una colonia en PDF
- Mapa con la ubicación de las colonias

---

## 6. Mejoras futuras

- Notificaciones para recordar desparasitaciones pendientes
- Historial médico completo de cada gato
- Sistema de usuarios y login para que cada voluntario tenga su cuenta
- Aplicación móvil
- Colaboración entre varias colonias o asociaciones

---

## 7. Tecnologías

- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: Express (Node.js)
- **Almacenamiento**: LocalStorage en el navegador para los datos del usuario
